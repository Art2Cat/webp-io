# webp-io

the webp-io is convert normal image and webp file.

## Usage

**Maven dependency**

```xml
<dependency>
    <groupId>io.github.biezhi</groupId>
    <artifactId>webp-io</artifactId>
    <version>0.0.1</version>
</dependency>
```

**API**

```java
String src = "a.webp";
String dest = "a.png";
WebpIO.toNormalImage(src, dest);
```
